# Generative AI Research – Project Repository

This repository contains all research materials, data, scripts, and documentation for the study:  
**“Using Generative AI Tools – Boon or Bane?”**

---

##  Project Structure


Each folder contains categorized materials such as articles, scripts, raw/cleaned data, interviews, reports, and media files.

---

##  How to Navigate

- **Literature_Review**: Background materials including journal articles and books.  
- **Quantitative_Analysis**: Survey data (raw + cleaned), scripts, and reports.  
- **Qualitative_Analysis**: Interview transcripts, consent forms, insights, and visualisations.  
- **Drafts_and_Reports**: Research proposal drafts and final outputs.  
- **Additional_Materials**: Photos, info sheets, and other media.  
- **Documentation_Drafts**: Notes and drafts for project documentation.
